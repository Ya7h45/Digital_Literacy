Project Overview

This project is a Digital Literacy Portfolio created as part of the CSE0001 course. The objective of this project is to develop essential digital skills required for academic and professional success. As a Student Digital Ambassador, the project focuses on building awareness about digital tools, safe internet practices, professional communication, and cybercrime prevention.

The project consists of five tasks, each aligned with a module of the course. These tasks include creating an infographic, building a digital portfolio, exploring coding platforms, practising email etiquette, and studying cybercrime awareness. All deliverables are organized in this repository along with a detailed project report.

Module Summaries
Module 1 – Digital Literacy Awareness

In this module, I created an infographic using Canva to explain the concept of digital literacy. It highlights key areas such as safe internet practices, useful tools for students, and maintaining a professional online presence.

Module 2 – Student Digital Portfolio

This module focuses on building a professional online presence. I created/updated profiles on platforms like GitHub, LinkedIn, and ResearchGate. These platforms help in showcasing skills, networking, and academic growth.

Module 3 – Coding & Collaboration Platforms

In this module, I explored platforms like HackerRank for coding practice and Google Forms for collaboration. I completed a beginner-level coding challenge and created a digital literacy quiz using Google Forms.

Module 4 – Email Etiquette & Communication

This module emphasizes professional communication. I drafted formal emails (assignment extension and internship request) and created a checklist of do’s and don’ts for responsible social media usage.

Module 5 – Cybercrime Awareness

In this module, I studied cybercrime (such as UPI fraud/phishing), created a case study, and developed a prevention checklist to promote safe online behavior among students.
